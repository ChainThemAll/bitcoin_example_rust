# 简单版比特币区块链网络

这是一个用Rust编写的简单版比特币区块链网络。

## 项目介绍

这个项目是一个简化版的比特币区块链网络，它实现了区块链的基本功能，包括创建区块、计算区块哈希、工作量证明、验证区块链的完整性以及进行交易等。

## 技术栈

- **语言**: Rust
- **库**:  serde、hex、anyhow、hex、sled、once_cell、tokio、tracing、ed25519、rand、

## 功能

- 创建区块
- 工作量证明
- 验证区块链完整性
- 进行交易
- p2p网络

## 如何运行

1. 克隆仓库到本地
2. 安装Rust和Cargo
3. 在项目根目录下运行 `cargo run`
